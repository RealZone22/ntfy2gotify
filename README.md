## Installation

You can download the latest release from the [releases page](https://github.com/RealZone22/ntfy2gotify/releases).
Alternatively, you can run it using Docker:

```bash
docker run -d \
  --name ntfy2gotify \
  -v ./config.json:/app/config.json \
  ghcr.io/realzone22/ntfy2gotify:latest
```
